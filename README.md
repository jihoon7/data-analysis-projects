## 주요 프로젝트

### · 문서 기반 검색 챗봇 (LLM/RAG)
GPT, ChromaDB, SentenceTransformers를 활용해  
내부 문서를 의미 기반으로 검색하고 답변하는 시스템을 구현했습니다.  
검색 정확도는 96%, 응답 속도는 약 40% 개선되었습니다.

### · 한국어 감정 분석 모델
KoBERT 기반으로 감정 분류 모델을 만들고, 전처리 자동화를 추가해  
정확도 93.8%, F1-score 0.91을 달성했습니다.

### · 공정 이상 탐지
센서 데이터를 기반으로 Isolation Forest와 Autoencoder를 조합하여  
오탐률을 40% 이상 줄였습니다.

### · 시계열 생산량 예측
LSTM과 Prophet을 함께 사용해 예측 모델을 만들었고,  
RMSE를 기존 대비 42% 줄였습니다.

### · 고객 이탈 예측
XGBoost, LightGBM 기반으로 모델을 구성해  
Precision 0.88, Recall 0.93의 결과를 얻었습니다.

### · 금융 문서 LLM QA 실험
은행 약관·상품 설명서를 기반으로 LLM QA 테스트를 진행했습니다.  
OCR 정제, 텍스트 구조화, 프롬프트 설계 등을 수행하며  
문서 기반 QA의 정확도에 영향을 주는 요소들을 분석했습니다.

### · 골프 센서 분석

스윙 패턴 분석 실험을 진행했습니다.
