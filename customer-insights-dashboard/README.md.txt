# Customer Insights Dashboard

## 프로젝트 설명
고객 유입 경로와 상담 데이터를 분석하여 Power BI 대시보드를 제작했습니다. 데이터를 시각적으로 표현해 보고 효율성을 높였습니다.

## 주요 기술
- 데이터 분석: Python, Pandas
- 데이터 시각화: Power BI

## 주요 성과
- 보고 효율성 20% 이상 증가
- 고객 데이터 기반 전략적 의사결정 지원

## 실행 방법
1. Power BI에서 `dashboard.pbix` 파일 열기
2. 데이터 업데이트 및 대시보드 확인

## 사용 기술 및 도구
- Python, Power BI
