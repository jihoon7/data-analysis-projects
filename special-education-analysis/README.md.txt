# Special Education Data Analysis

## 프로젝트 설명
특수교육 데이터를 분석하여 주요 사용자 트렌드를 도출하고 정책 제안을 지원했습니다.

## 주요 기술
- 데이터 수집: 정부 기관 협력
- 데이터 분석: 코호트 분석, Pandas

## 주요 성과
- 데이터 기반 정책 제안
- 생산성 20% 향상
- 사용자 행동 데이터를 기반으로 주요 트렌드 도출

## 실행 방법
1. **데이터 준비**: `data/` 폴더에 데이터 업로드
2. **분석 실행**: `python cohort_analysis.py`

## 사용 기술 및 도구
- Python, Pandas
